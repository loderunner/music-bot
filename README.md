# music-bot
A bot for the Music flow of Lima's Flowdock. Warns you when you posrt too often.

## Getting started

```bash
git clone git@github.com:loderunner/music-bot.git
cd music-bot
virtualenv env
. env/bin/activate
pip install -r requirements.txt
vi config.json
# Set your configuration
python music-bot.py
```
